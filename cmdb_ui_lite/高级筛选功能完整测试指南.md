### 1. `visibleFilterTags` 计算属性 ([general-model/index.vue#L430-L438](src/views/general-model/index.vue#L430-L438))
**修复前问题**: 数组类型的筛选值会被错误过滤掉
**修复后逻辑**:
```javascript
visibleFilterTags() {
  return this.filterTags.filter(tag => {
    const value = tag.value
    if (Array.isArray(value)) {
      return value.length > 0  // 数组类型检查长度
    }
    return value !== null && value !== undefined && String(value).trim().length > 0
  })
},
```

### 2. `handleAdvancedFilterSearch` 方法 ([general-model/index.vue#L1110-L1134](src/views/general-model/index.vue#L1110-L1134))
**修复前问题**: 依赖 `rawConditions`，可能导致标签不显示
**修复后逻辑**: 直接基于 `conditionMap` 构建标签，确保响应式更新

### 3. `restoreStateFromUrl` 方法 ([general-model/index.vue#L1008-L1025](src/views/general-model/index.vue#L1008-L1025))
**修复前问题**: 从 URL 恢复时可能无法正确构建标签
**修复后逻辑**: 同样基于 `conditionMap` 正确构建标签
